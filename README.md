# Arduino
아두이노를 사용한 피지컬 컴퓨팅 실습 Repository 입니다.

# Minipiano
* 4개의 swith 를 사용하여 8개의 음계를 내는 미니 피아노를 제작한 프로젝트로,
* 기존의 8개의 스위치 핀을 사용한다면 필요한 스위치와 점파선의 수를 절감해보자는 단순한 니즈에서 출발하였습니다.
